---

### 🔬 Featured work

**[Designing a LAMP-PCR Biosensor for Heavy-Metal Detection — Gowanus Canal](https://github.com/karolinczi/gowanus-lamp-biosensor-thesis)**
My M.S. thesis. A low-cost, field-deployable biosensor for heavy-metal contamination at the Gowanus Canal Superfund site — metagenomics to find heavy-metal-resistance genes → LAMP-PCR primer design & wet-lab validation (detected a cobalt-zinc-cadmium *czc* resistance gene) → community-centered design with the Gowanus Dredgers.

**[Savannah River Site Metagenomics](https://github.com/karolinczi/savannah-river-metagenomics)**
Shotgun-metagenomic analysis of a uranium/nickel contamination gradient — MetaPhlAn 4 taxonomy + HumAnN functional profiling, correlating heavy-metal-resistance gene abundance with measured metal concentrations.

**[Splice-Junction Sequence Analysis](https://github.com/karolinczi/2026-festival-of-genomics-workshop/blob/main/notebooks/splice_junction.py)**
Recovering the GT-AG splice-site rule directly from base frequencies and training a classifier on DNA sequence windows (Festival of Genomics 2026 workshop).
